Built using the MERN stack the application bridges the gap between complex market data and intuitive user interaction, serving and comprehensive tool for and real-time market analysis
# 📈 StockX | Full-Stack Stock Trading Platform
**A Zerodha-inspired trading ecosystem built for high-performance and seamless UI/UX.**

StockX is a comprehensive MERN-stack application that replicates the core functionalities of modern trading platforms like Zerodha Kite. It combines a minimalist, "distraction-free" design with robust backend logic for portfolio management and market tracking.

---

## ✨ Features

### 🎨 UI/UX Excellence
* **Kite-Inspired Interface:** Clean, white-and-blue aesthetic (or dark mode) focusing on usability.
* **Responsive Dashboard:** Fully optimized for Mobile, Tablet, and Desktop views.
* **Interactive Charts:** Real-time data visualization using **Chart.js / Recharts**.
* **Micro-interactions:** Smooth transitions and hover effects for a premium feel.

### ⚙️ Core Functionalities
* **Market Watchlist:** Add/remove stocks to track real-time price movements.
* **Order Execution:** Simulate 'Market' and 'Limit' orders for buying/selling stocks.
* **Portfolio Management:** Track holdings, calculate average price, and monitor live P&L (Profit & Loss).
* **Funds Portal:** Manage virtual balance to simulate real-world trading constraints.
* **Secure Auth:** JWT-based user authentication and protected API routes.
